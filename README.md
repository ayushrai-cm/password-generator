# Password Generator (Python Project)

A secure and customizable command-line Password Generator built in Python. It helps users create strong, random passwords with different character combinations to improve security.

## Features

-Generate secure random passwords
-Customizable length (4–64 characters)
-Option to include:
-Uppercase letters
-Digits
-Symbols
-Generates multiple passwords at once
-Input validation for better user experience

## How to Run

git clone https://github.com/ayushrai-cm/password-generator.git
cd password-generator
python password_generator.py

## Tech Used

-Python
-secrets module (for secure randomness)
-string module
